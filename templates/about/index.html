어르신 비상웹캠

어른신 비상앱은 시간 설정 및 동작 설정을 하여 이상이 있을때 자동적으로 119및 경찰서로 바로 신고를 하고 보호자에
비상 알림을 알리는 시스템입니다.

Get Started

웹 앱 : 웹 앱을 통해 각종 다양한 기능을 통한 서비스를 사용할 수 있게 하는 GUI 구현 회원가입 하는 과정에서 개인정보를 입력하여 SOS 기능을 보호자에게 전송하여 웹 앱의 알림 또는 메시지로 인증 메시지를 전송하도록 해주는 형태로 구현 예정
CNN 모델을 사용하여 쓰러진 사람을 감지하고 구별하는 시스템 구현
CCTV 캠을 사용하여 실시간 모니터링 및 응급조치 알림 기능 제공
캠을 통해 쓰러진 긴급한 노인을 감지하여 웹 앱과 통신을 하여 보호자에게 전송하는 기능을 구현하였습니다.
